西 3章 11节 注1
　　“此”是指10节的新人，“在此”指在新人里。
西 3章 11节 注2
　　在新人里不仅没有天然的人，连天然的人存在的可能和地位都没有。
西 3章 11节 注3
　　希利尼人是要哲学的智慧，犹太人是要神迹。（林前一22。）
西 3章 11节 注4
　　受割礼的，指遵守犹太宗教仪式的人；未受割礼的，指不理会犹太宗教的人。
西 3章 11节 注5
　　化外人是未开化的人。
西 3章 11节 注6
　　西古提人被视为最野蛮的人。
西 3章 11节 注7
　　卖身为奴的人。
西 3章 11节 注8
　　从为奴得自由的人。
西 3章 11节 注9
　　在新人里只有基督有地位。祂是新人一切的肢体，也在一切的肢体之内。在新人里祂是一切。实际上，祂就是新人，就是祂的身体。（林前十二12。）在新人里祂是中心，也是普及，是构成新人的成分，是新人里一切的一切。
西 3章 11节 注10
　　“一切，”指组成新人的众肢体。
